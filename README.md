# Телефонный справочник

Веб-приложение корпоративного телефонного справочника с публичной страницей поиска и административной панелью для управления контактами.

---

## Стек технологий

| Слой | Технология |
|---|---|
| Язык | Python 3 |
| Веб-фреймворк | FastAPI |
| ASGI-сервер | Uvicorn |
| База данных | SQLite 3 |
| Шаблоны | Jinja2 |
| Стили | CSS (без фреймворков) |

---

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <url-репозитория>
cd phonebook
```

### 2. Создать и активировать виртуальное окружение

```bash
python -m venv venv

# Linux / macOS
source venv/bin/activate

# Windows
venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить приложение

```bash
# Windows
python main.py

# Linux / macOS
python3 main.py
```

После запуска приложение будет доступно по адресу: <http://127.0.0.1:8000> (или по IP сервера, если запущено удалённо)

> База данных `phonebook.db` создаётся автоматически при первом запуске и заполняется тестовыми данными.

## Учётные данные администратора по умолчанию

| Поле | Значение |
|---|---|
| URL панели | `/admin/login` |
| Логин | `admin` |
| Пароль | `admin123` |

> ⚠️ Смените пароль после первого входа через раздел «Сменить пароль» в административной панели.

---

## Структура базы данных

### Таблица `admins`

Хранит учётные записи администраторов.

| Колонка | Тип | Описание |
|---|---|---|
| `id` | INTEGER PK | Идентификатор |
| `username` | TEXT UNIQUE | Логин администратора |
| `password` | TEXT | SHA-256 хеш пароля |

---

### Таблица `departments`

Хранит отделы организации.

| Колонка | Тип | Описание |
|---|---|---|
| `id` | INTEGER PK | Идентификатор |
| `name` | TEXT | Название отдела |
| `sort_order` | INTEGER | Порядок сортировки на странице (по умолчанию `0`) |

---

### Таблица `contacts`

Хранит записи сотрудников. Связана с `departments` через внешний ключ.

| Колонка | Тип | Описание |
|---|---|---|
| `id` | INTEGER PK | Идентификатор |
| `department_id` | INTEGER FK | Ссылка на отдел (`departments.id`) |
| `position` | TEXT | Должность сотрудника |
| `full_name` | TEXT | ФИО сотрудника |
| `internal_phone` | TEXT | Внутренний номер телефона |
| `external_phone` | TEXT | Внешний номер телефона |
| `office` | TEXT | Номер кабинета |

При удалении отдела все связанные контакты удаляются автоматически (`ON DELETE CASCADE`).
